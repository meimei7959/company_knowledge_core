# Agent Runners

This directory is the central registry area for external Agent Ring runner records.

Agent Ring itself is not implemented in this repository. Each runner record describes a distributed computer connected to the central processor.

Runner records should describe:

- runnerId;
- host type;
- status and heartbeat;
- registered Agents;
- capabilities;
- project and repository access;
- data scopes;
- resource profile;
- current load;
- Ring version.

Do not store local secrets here.
