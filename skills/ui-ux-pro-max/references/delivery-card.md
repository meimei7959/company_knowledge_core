# UI/UX Pro Max Delivery Card

## Use

Use as the quality gate for front-end user experience.

## Checks

- Four states: loading, success, error, empty.
- Operation feedback: click, loading, success, failure.
- Responsive behavior: mobile/narrow screen is usable.
- Readability: user sees business meaning, not internal codes.
- Progressive disclosure: technical details are hidden until needed.

## Stop

- Main UI is not understandable.
- No feedback after actions.
- Error state lacks next action.
- Internal ids or raw statuses appear as primary content.
