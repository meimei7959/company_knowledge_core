# Mature Skill Package Reference

Use this reference when creating or updating a production skill.

## Required Files

- `SKILL.md`: concise trigger and workflow.
- `references/delivery-card.md`: job-to-be-done, source references, role binding, failure routing.
- `templates/output-template.md`: output shape an Agent can fill.
- `examples/quality-example.md`: example of acceptable evidence-backed output.

## Required Output Qualities

- Chinese by default.
- Source evidence included when facts or conclusions are produced.
- Explicit assumptions.
- Clear failure route when input is missing or unsafe.
- No secret values.
- No hidden approval wait; blocked work must surface status and owner.
