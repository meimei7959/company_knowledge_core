# Agents

The company knowledge core is maintained by three core Agents:

- [Company Knowledge Core Knowledge Engineering Agent](../../agents/agent.company-knowledge-core.knowledge-engineering.md): source material reading, evidence packet building, structured knowledge draft, and task writeback.
- [Knowledge Steward Agent](../../agents/core/agent.knowledge-steward.md): rules, boundaries, registry classification, and structure.
- [Knowledge Review Agent](../../agents/core/agent.knowledge-review.md): quality gate, source evidence, conflict detection, and approval routing.
- [Knowledge Ops Agent](../../agents/core/agent.knowledge-ops.md): connector, gateway, permission checks, audit, evaluation, sync, and recovery.

Operating model:

- [Knowledge Core Agent Team](../../docs/agent-team/knowledge-core-agent-team.md)

- `agent.codex.local`: local Codex builder.
- `agent.antigravity.local`: local Antigravity builder.
- [公司项目经理 Agent](../../agents/agent.company.project-manager.md) - 负责项目初始化、任务编排、节点验收、风险跟踪、通知与跨岗位协同。
- [查知识 Agent](../../agents/agent.company.knowledge-query.md) - 面向员工和其他 Agent 检索已审核知识，返回答案、来源、置信度和缺口。
- [产品经理 Agent](../../agents/agent.company.product-manager.md) - 负责需求澄清、用户场景、PRD、验收标准、竞品和市场分析。
- [设计 Agent](../../agents/agent.company.design.md) - 负责信息架构、交互流程、界面方案、可用性和设计交付。
- [研发 Agent](../../agents/agent.company.development.md) - 负责技术方案、代码实现、调试、测试和工程交付。
- [测试 Agent](../../agents/agent.company.test.md) - 负责测试计划、用例、自动化验证、缺陷复现和质量门禁。
- [运营 Agent](../../agents/agent.company.operations.md) - 负责运营计划、内容/活动执行、反馈收集、增长实验和运营复盘。
