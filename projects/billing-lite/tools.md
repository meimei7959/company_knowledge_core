# Billing Lite Tools

## Registered Project Tools

- `tool.zhenzhi-knowledge`: project/task/source/result/audit record management.

## Tool Boundary

Payment provider credentials, webhook secrets, Apple / Google API keys, PSP keys, certificates, and tokens must be stored only through approved secret management. They must not be written into project files, task files, TaskResult, or knowledge drafts.

Future tool additions require Tool Owner approval before agent use.
