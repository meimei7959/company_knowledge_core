# Tools Index

ToolAsset files describe reusable team tools.

Tool code lives in Git or its runtime. This knowledge bundle stores metadata, ownership, risk, permissions, usage notes, and review status.

Each ToolAsset file must include:

- `type: ToolAsset`
- `toolId`
- `owner`
- `repoRef`
- `entrypoint`
- `status`
- `riskLevel`
