# Agents Index

Agents represent local or hosted AI workers registered to the knowledge core.

Each Agent file must be OKF-compatible and include:

- `type: Agent`
- `agentId`
- `owner`
- `aiTool`
- `allowedProjects`
- `allowedTools`
- `allowedKnowledgeScopes`
- `status`
