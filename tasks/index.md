# Tasks

Company-level task cards live here when they are not scoped to a single project directory.

Project-specific tasks may live under `projects/<project-id>/tasks/`.

Use `ProjectTask` for general project work and `KnowledgeTask` for source-material-to-knowledge work.

Tasks are scheduled by the central processor and executed by external Agent Ring runners.
